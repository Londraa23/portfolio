# Portfolio de Pablo Estévez

Bienvenido a mi portfolio personal. Aquí muestro mis proyectos, habilidades y experiencia como desarrollador Full-Stack y administrador de sistemas. Esta web ha sido diseñada y desarrollada con un enfoque limpio, moderno y funcional utilizando tecnologías actuales como Next.js, Tailwind CSS y Lucide Icons.

## 👤 Sobre mí

Soy Pablo Estévez, estudiante de segundo año del ciclo de **Sistemas Microinformáticos y Redes (SMR)**. Tengo 18 años y experiencia tanto en desarrollo como en administración de sistemas. Me apasiona la tecnología, el aprendizaje continuo y construir soluciones prácticas mediante el software.
